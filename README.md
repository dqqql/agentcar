# 智能座舱出行规划 Agent

## 项目说明

本项目用于实现一个面向智能座舱场景的出行规划 Agent。
用户通过自然语言输入出行需求，系统解析需求后，结合偏好和推荐逻辑生成景点、酒店、餐饮和行程规划结果。

当前项目处于刚开工阶段，已完成 `T0 项目初始化`，下一步会先学习 FastAPI 的基本用法，再进入 `T1 后端启动`。

## 当前开发策略

- 先做 MVP 闭环，不追求一次做全
- 先用 mock 数据打通流程，再考虑真实 API
- 先做后端最小主链路，再接前端演示页
- 每一轮都要有可截图、可录屏、可向老师汇报的结果

## 当前 MVP 目标

MVP 目标是跑通这一条链路：

`自然语言输入 -> 需求解析 -> 推荐结果 -> 行程生成 -> 页面展示`

本阶段暂不优先处理：

- 真实座舱接入
- 语音输入
- 深度学习推荐
- 多轮复杂对话
- 实时价格查询
